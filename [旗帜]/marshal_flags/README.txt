HOW TO INSTALL VORP
___________________________________________________
DEPENDENCIES: 
[https://github.com/VORPCORE/VORP-Inventory]
[https://github.com/VORPCORE/VORP-Core]
[https://github.com/VORPCORE/VORP-Character]

1. Insert the itemsVORP.sql file into your database called itemsVORP
2. Copy and paste all images into vorp_inventory/html/img/items
3. Now go to config.lua inside of marshal_flags and change Config.framework to 'VORP'
4. Toggle and set any settings or text for your server if needed.
5. Make sure "ensure marshal_flags is set in your server.cfg or resources.cfg, etc"

HOW TO INSTALL REDEM-RP
___________________________________________________
DEPENDENCIES:
[https://github.com/kanersps/redem]
[https://github.com/RedEM-RP/redem_roleplay]
[https://github.com/RedEM-RP/redemrp_identity]
[https://github.com/RedEM-RP/redemrp_inventory]
[https://github.com/RedEM-RP/redemrp_respawn]

1. Copy and paste the contents inside of itemsREDEMRP.lua into redemrp_inventory inside of the Config.items list
2. Copy and paste all images into redemrp_inventory/html/items
3. Now go to config.lua inside of marshal_flags and change Config.framework to 'REDEMRP'
4. Toggle and set any settings or text for your server if needed.
5. Make sure "ensure marshal_flags is set in your server.cfg or resources.cfg, etc"

HOW TO INSTALL STANDALONE
___________________________________________________
1. Simply set framework to 'OTHER'
2. if you want to use a specific flag Config.setDefaultType and Config.defaultFlagType are useful
3. Refer to type in Config.items
4. Make sure "ensure marshal_flags is set in your server.cfg or resources.cfg, etc"

TOOLS AND TIPS
___________________________________________________
- Default settings should be good for most servers as long as your framework is selected correctly.
- Use the most up to date versions of RedM!! It is good practice when running a server.
- Config.debug is useful if you need support, etc. Can also diagnose issues on your server!
- Refer to Config.lua for more detailed descriptions and help!
- If you need additional assistance please go to the support discord [https://discord.gg/JDNYnbww9q]

如何安装 VORP
___________________________________________________
依赖项：
[https://github.com/VORPCORE/VORP-Inventory]
[https://github.com/VORPCORE/VORP-Core]
[https://github.com/VORPCORE/VORP-Character]

1. 将 itemsVORP.sql 文件插入到名为 itemsVORP 的数据库中。
2. 将所有图像复制并粘贴到 vorp_inventory/html/img/items 文件夹中。
3. 现在打开 marshal_flags 文件夹中的 config.lua，将 Config.framework 更改为 'VORP'。
4. 如有需要，请切换并设置服务器的任何设置或文本。
5. 确保在服务器的 server.cfg 或 resources.cfg 中设置了 "ensure marshal_flags"。

如何安装 REDM-RP
___________________________________________________
依赖项：
[https://github.com/kanersps/redem]
[https://github.com/RedEM-RP/redem_roleplay]
[https://github.com/RedEM-RP/redemrp_identity]
[https://github.com/RedEM-RP/redemrp_inventory]
[https://github.com/RedEM-RP/redemrp_respawn]

1. 将 itemsREDEMRP.lua 文件中的内容复制并粘贴到 Config.items 列表中的 redemrp_inventory。
2. 将所有图像复制并粘贴到 redemrp_inventory/html/items 文件夹中。
3. 现在打开 marshal_flags 文件夹中的 config.lua，将 Config.framework 更改为 'REDEMRP'。
4. 如有需要，请切换并设置服务器的任何设置或文本。
5. 确保在服务器的 server.cfg 或 resources.cfg 中设置了 "ensure marshal_flags"。

如何安装独立版
___________________________________________________
1. 简单地将 framework 设置为 'OTHER'。
2. 如果您想使用特定标志，Config.setDefaultType 和 Config.defaultFlagType 非常有用。
3. 参考 Config.items 中的类型。
4. 确保在服务器的 server.cfg 或 resources.cfg 中设置了 "ensure marshal_flags"。

工具和提示
___________________________________________________
- 默认设置对于大多数服务器来说应该足够好，只要您选择了正确的框架。
- 使用最新版本的 RedM！这是运行服务器的良好实践。
- 如果需要支持等方面的帮助，Config.debug 很有用。也可以用于诊断服务器上的问题！
- 有关更详细的描述和帮助，请参考 Config.lua。
- 如果需要额外的帮助，请访问支持 Discord [https://discord.gg/JDNYnbww9q]。
